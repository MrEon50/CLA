## 🌟 Co nowego w v10.4

### 1. 🌀 Model Triple-Friction (3D)
Zastąpienie płaskiego parametru tarcia trójosiowym modelem napięć:
- **$F_{aff}$ (Affective Friction)**: Napięcie emocjonalne i empatyczne.
- **$F_{cog}$ (Cognitive Friction)**: Napięcie poznawcze wywoływane przez paradoksy, trudność analizy i nieznane fakty.
- **$F_{axio}$ (Axiological Friction)**: Napięcie aksjologiczne wywoływane przez konflikty z instynktami i weryfikację spójności z "prawdą".

### 2. 🧬 Rozwoj i Inkubacja
- **DevelopmentEngine (Silnik Rozwoju)**: Bot rośnie kognitywnie z każdą interakcją. Posiada wiek kognitywny i etapy rozwoju (od Primal po Transcendent).
- **Thought Incubator**: Myśli, które potrzebują czasu, dojrzewają w tle świadomości, pozwalając na proaktywne powracanie do tematów.

## 🚀 Szybki Start (Quick Start)

### Wymagania:
- Python 3.10+
- [Ollama](https://ollama.com/) zainstalowana i uruchomiona.
- `ollama pull mxbai-embed-large` (dla Pamięci Asocjacyjnej).
- `ollama pull qwen3.5:9b` (przykładowy model do rozmowy lub chmurowy).

```bash
# 1. Instalacja zależności
pip install -r requirements.txt

# 2. Uruchomienie Systemu
python clatalkie.py
```

## 🎮 Kluczowe Komendy

### 🧠 Interakcja Kognitywna
- `/think` - Konsolidacja myśli, budowa grafu konceptów i proces zapominania (decay).
- `/meditation [N]` - Sesja swobodnych myśli oparta na aktualnym stanie kognitywnym.
- `/status` - Podgląd aktualnej "fizjologii" (V, F, S) i B-Index.
- `/scan <ścieżka> [--learn]` - Analiza lokalnych plików za pomocą modelu - tryb `--learn` włącza informacje trwale do kognitywnej bazy wiedzy.
- `/chain <N> [temat]` - Generowanie logicznych ciągów rozumowania.
- `/memory` - Zrzut aktualnej architektury DNA, w tym bazowych pojęć w pamięci trwałej i konstelacji pojęciowych.
- `/reindex` - Ponowna indeksacja bazy wektorowej używanej w skojarzeniach, przydatna w razie problemów z RAG.
- `/gender <typ>` - Zmiana postrzeganej przez Clatalkie tożsamości płciowej (np. masculine, feminine, neutral, fluid).
- `/model <prompt>` -  Strzał próbny do bezpośredniego zapytania na model bazowy.
